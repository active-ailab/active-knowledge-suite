# Workspace Map

Use this file when the task needs a quick mental model of the top-level active-knowledge tree.

## Top-Level Areas

- `doc/`: development document
- `knowledge-sources/`: source documents that should be remembered by `active-knowledge-server`
- `active-knowledge-graph/`: active knowledge skill to use active-knowledge-server
- `active-knowledge-server/`: active knowledge mcp server
- `active-knowledge-workspace/`: active knowledge skill to mental active workspace tree
